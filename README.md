# jronrpc

A Go project.

## Getting Started

### Prerequisites

- Go 1.19 or later

### Installation

```bash
git clone jronrpc
cd jronrpc
go mod tidy
```

### Usage

```bash
go run main.go
```

### Build

```bash
go build -o jronrpc
./jronrpc
```
